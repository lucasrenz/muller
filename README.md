# Novo Sistema

Repositorio unico, preparado para um deploy so, com os sistemas separados por pasta.

Instrucoes rapidas:

- Instalar dependencias: `npm install`
- Rodar em dev: `npm run dev`

Organizacao desta copia:

- `src/sistemas/financeiro` cobre o painel financeiro/cartao
- `src/sistemas/denuncias` cobre o sistema de denuncias
- `src/sistemas/vagas` cobre o RH/vagas
- `src/sistemas/pagina-principal` cobre a entrada inicial
- `docs/financeiro`, `docs/denuncias` e `docs/rh` guardam os arquivos especificos de cada sistema
